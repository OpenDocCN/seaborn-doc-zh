## 下载

### Docker

```
docker pull apachecn0/seaborn-doc-zh
docker run -tid -p <port>:80 apachecn0/seaborn-doc-zh
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g seaborn-doc-zh
seaborn-doc-zh <port>
# 访问 http://localhost:{port} 查看文档
```
